# Resume Bullet Points — IPL Data Analytics & Match Winner Prediction System

1. Engineered an end-to-end IPL analytics pipeline in Python (Pandas, NumPy) processing
   96,000+ ball-by-ball records and 400+ matches, including data cleaning, deduplication,
   team-name standardization, and feature engineering (CRR, RRR, run-rate metrics).

2. Designed and built 24+ data visualizations using Matplotlib and Seaborn (heatmaps,
   correlation matrices, box plots, scatter plots) to analyze player performance, team
   win rates, venue trends, and season-wise statistics.

3. Developed and benchmarked Logistic Regression, Random Forest, and XGBoost classifiers
   for real-time IPL match-winner prediction, achieving up to 99.9% accuracy and 1.00
   ROC-AUC with XGBoost, evaluated via Precision, Recall, and F1-Score.

4. Designed an interactive 5-page Power BI dashboard (Overview, Player Analytics, Team
   Analytics, Venue Analytics, Match Prediction) with DAX measures for win percentage,
   head-to-head comparisons, and toss-impact analysis.

5. Deployed a production-ready Streamlit web application with a custom dark/IPL-themed UI,
   enabling live match-winner predictions and win-probability visualizations via Plotly,
   hosted on Streamlit Cloud and Render.
